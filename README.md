# Fitness and Health Centre

## Introduction

This project is a Gym Management System developed using Oracle 10g Express Edition.

The system manages:

- Members
- Trainers
- Nutritionists
- Workout Sessions
- Equipment
- Attendance
- Invoices
- Diet Plans
- Feedback

## Objectives

- Design relational database
- Create relationships
- Perform SQL queries
- Create views
- Implement triggers

## Technologies Used

- Oracle 10g Express Edition
- SQL
- Oracle SQL Command Line

## Main Features

- Member Management
- Trainer Management
- Attendance System
- Billing System
- Workout Sessions
- Diet Plans
